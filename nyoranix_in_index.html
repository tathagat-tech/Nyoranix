Hosting checklist

  1. Create a GitHub repo (see steps below)
  2. Add this index.html to the repo root
  3. Enable GitHub Pages (branch: main, folder: / (root))
  4. If using custom domain nyoranix.in, add a CNAME file with that domain and configure DNS records

Notes

GitHub Pages will provide HTTPS automatically after DNS configuration propagates.

Small deploy guide (commands) 
# in terminal
mkdir nyoranix-site
cd nyoranix-site
git init
# create index.html (save this file)
# then
git add index.html
git commit -m "Initial site"
# create repo on GitHub and add remote (replace URL)
git remote add origin https://github.com/YOURUSERNAME/nyoranix-site.git
git branch -M main
git push -u origin main

After pushing, go to your repository → Settings → Pages and set Source to main / root. Add a CNAME file if you want the custom domain.
